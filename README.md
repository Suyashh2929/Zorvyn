🚀 Zorvyn Finance: Institutional Asset Dashboard
Zorvyn Finance is a high-performance, responsive Fintech dashboard built with React and Tailwind CSS. Originally conceptualized as a scheduling platform, it has been engineered into a sophisticated wealth management terminal that allows users to track global assets, manage portfolios, and analyze market sectors in real-time.

💎 Key Features
Pro-Trader Dark Theme: A premium "Glassmorphism" UI using deep navy tones, emerald accents, and backdrop blurs for an institutional feel.

Live Market Intelligence: Real-time data visualization for stocks, crypto, and traditional assets.

Dynamic Portfolio Management: Track active positions, calculate P/L (Profit/Loss), and manage institutional risk appetites.

Secure Auth Portal: High-security login and registration flow featuring "Access Key" protocols.

Mobile-First Engineering: Fully responsive navigation and data tables optimized for iPhone and Android devices.

Institutional Identity: Comprehensive user profiles including KYC status, UID tracking, and 2FA security toggles.

🛠️ Tech Stack
Frontend: React.js (Hooks, Context API, Router)

Styling: Tailwind CSS (Utility-first CSS, Responsive Design)

Icons/Assets: Custom Fintech Asset Pack

Build Tool: Vite (Ultra-fast HMR)

📂 Project Structure
Based on our institutional architecture, the project is organized as follows:

Components
Navbar: Centered navigation with glassmorphism effects and secure logout logic.

Header: Dynamic hero section with market status indicators.

TopAssets: High-density grid showcasing trending institutional assets.

SpecialityMenu: Sector-based market navigation (Tech, Crypto, Energy).

Footer: Slimmed regulatory footer with legal disclosures and SEC/SIPC placeholders.

Pages
Home: The main dashboard overview.

Screener: Deep-dive transaction and asset history.

Appointments: Asset Intelligence page with order execution terminals.

MyProfile: Secure identity and KYC management hub.

MyAppointments: Active portfolio positions and real-time tracking.

Login/Signup: Secure entry portal.
